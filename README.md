# GrokImgae
GrokImgae项目 配合grok2api使用 很爽的
app.JS
里面修改这几个参数
const CONFIG = {
    defaultBaseUrl: '逆向程序:端口',
    defaultApiUrl: 'https://逆向程序:端口/v1/chat/completions',
    defaultModel: 'grok-4.1'
};
在index.html 里面也要搜一下进行修改  也可以在前端直接填写运行
GROK支持生图的模型均可使用 采用base64编码进行传输
